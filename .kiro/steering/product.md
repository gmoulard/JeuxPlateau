# Product Overview

## Jeux de Plateau - 100% Vibecodé

A Progressive Web App (PWA) for playing classic board games, entirely developed using AI-assisted development.

### Core Features
- **6 Classic Games**: Tic-Tac-Toe (Morpion), Checkers (Dames), Chess (Échecs), Backgammon (Tavli), Ludo (Petits Chevaux), Abalone
- **PWA Support**: Installable on mobile devices with offline functionality
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Local Multiplayer**: 2-4 players depending on the game
- **French Interface**: All text and help content in French
- **Customization**: Color themes for boards and pieces
- **Game History**: Local storage of game sessions
- **Help System**: Contextual help for each game with official rules links

### Target Audience
French-speaking users looking for classic board games on mobile and web platforms.

### Development Philosophy
This project is a showcase of **AI-Assisted Development** using conversational programming and LLM-driven development workflows. Every feature has been developed through natural language prompts and AI code generation.
